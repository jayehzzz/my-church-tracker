# Workflows Directory

Workflows are step-by-step guides for common development tasks. Use them via slash commands.

## Available Workflows

| Command | File | Purpose |
|---------|------|---------|
| `/add-page` | [add-page.md](./add-page.md) | Create a new route/page |
| `/deploy` | [deploy.md](./deploy.md) | Deploy to Vercel with Convex |
| `/environments` | [environments.md](./environments.md) | Switch between test/production |
| `/seed-database` | [seed-database.md](./seed-database.md) | Seed test data |
| `/testing` | [testing.md](./testing.md) | Run and write tests |

## Usage

Type the slash command (e.g., `/deploy`) and the AI will follow the documented steps.

## Turbo Annotations

- `// turbo` - Auto-run the next command step
- `// turbo-all` - Auto-run all command steps in the workflow
